### 💡 Projetos em Destaque | Featured Projects

#### 🌾 [Smart Agri-Management Platform](https://github.com/kelvinnobuo/agri-smart-platform)
* **EN:** Web system integrated with Firebase and **Gemini 2.5 Flash API** to automate agricultural receipt (romaneio) data entry via OCR.
* **PT:** Sistema web integrado com Firebase e **API do Gemini 2.5 Flash** para automatizar o lançamento de romaneios agrícolas por OCR inteligente.
* **Impact:** Automates parsing of weight, moisture, and discounts directly from photos, saving entries in real-time.

#### 🗺️ [Agri-GIS & Cost Platform](https://github.com/kelvinnobuo/agri-gis-platform)
* **EN:** Cost tracking dashboard featuring dynamic GIS map overlays (Leaflet.js) to monitor chemical stock and plot-level farming costs.
* **PT:** Dashboard financeiro com mapas GIS dinâmicos (Leaflet.js) para controle de defensivos e custos de produção por talhão de plantio.

#### 🚜 [Massey Ferguson Harvest Visualizer](https://github.com/kelvinnobuo/harvester-visualizer)
* **EN:** Python tool to parse and render yield & moisture maps exported from Massey Ferguson 6690 harvesters.
* **PT:** Ferramenta Python para processar e renderizar mapas de produtividade e umidade exportados por colheitadeiras MF 6690.
